## Como contribuir?

Se por algum motivo esteja interessado em contribuir ou somente rodar na sua máquina, esses são os comandos:

+ Instale as dependências:

  ```bash
  pnpm i
  ```

+ Inicie em modo de desenvolvimento:

  ```bash
  pnpm dev
  ```

+ Crie a `build` do projeto:

  ```bash
  pnpm build
  ```
